# Active Directory Security Lab

## Overview

This repository documents my hands-on Active Directory Security Lab journey, covering domain administration, authentication mechanisms, offensive security techniques, detection strategies, and security hardening practices.

The goal of this project is to develop practical expertise in Active Directory Security from both Red Team and Blue Team perspectives while documenting the entire learning process.

---

## Lab Objectives

* Build and manage an Active Directory environment
* Understand AD architecture and authentication mechanisms
* Explore common attack techniques used by adversaries
* Learn detection and monitoring strategies
* Implement security hardening and mitigation controls
* Develop practical skills relevant to Active Directory security assessments and CRTP-level concepts
