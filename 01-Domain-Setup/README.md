# Domain Setup

## Objective

Build an Active Directory environment consisting of a Domain Controller and a domain-joined Windows client.

## Lab Architecture

### Domain Controller
- Windows Server 2022
- Active Directory Domain Services
- DNS Server

### Client Machine
- Windows 10
- Domain Joined
